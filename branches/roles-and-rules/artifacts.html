This page provides a list of the FHIR artifacts defined as part of this implementation guide.

Terminology: Value Sets

These define sets of codes used by systems conforming to this implementation guide.

Data Access Categories Value Set

Data Access Categories - the types of data that are associated with authorization to consult the data
Data Read Value Set

Data Read roles - the roles that are associated with authorization to consult the data

Terminology: Code Systems

These define new code systems used by systems conforming to this implementation guide.

Data Access categories

Data Access categories - the rolestypes of data that are associated with specifica authorization rules
Data Access Roles

Data Access roles - the roles that are associated with authorization to consult the data
